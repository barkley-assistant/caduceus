The task packet (`planning/caduceus-v1.0/tasks/4.2-reconcile-ambiguous-external-side-effects.md`) named `src/runtime/finalize.rs` (modified) and `src/state/checkpoints.rs` (modified) as the primary owned surfaces. The implementation places the equivalent helpers (`pr_exists`, `branch_exists`, `comment_exists` lookups) as methods on the existing `src/github/client.rs` struct instead of a new module. This is a smaller surface change and keeps related HTTP code colocated.
