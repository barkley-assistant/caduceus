## State/schema effects

- `QueueEntry` JSON schema now includes `generation: u32` (default 1 in all constructors).
- SQLite `queue_entries` table has `generation INTEGER NOT NULL DEFAULT 1`.
- JSON files with the old schema (no generation field) will fail to parse
  because `QueueEntry` uses `#[serde(deny_unknown_fields)]`. Migration
  (Task 3.3) sets generation=1 for all entries.
