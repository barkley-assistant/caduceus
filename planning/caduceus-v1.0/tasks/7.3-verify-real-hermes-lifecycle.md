# Task 7.3: Verify real Hermes lifecycle

## Outcome and required behavior

Provision an isolated disposable real Hermes Agent v0.18.2 gateway under a
temporary `$HERMES_HOME`. An external harness owns restart and scheduled
delivery observation; no operator home or gateway state is used.

- Owned surfaces: Hermes v0.18.2 lifecycle tests and evidence.
- Dependencies: 1.4, 2.1, 2.2, 7.1.
- Do not implement deferred v1.x work or modify the v0.1 planning tree.
- Preserve unrelated changes and the established public documentation voice.

## Primary tests
